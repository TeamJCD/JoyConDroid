# Joy-Con Droid

## Dual-Mode Operation

### Rooted Devices (Recommended)
- **Full Nintendo Switch Compatibility**: Uses HID Device API for complete controller emulation
- **All Features Supported**: Motion controls, rumble, LED indicators, NFC/amiibo support
- **Best Performance**: Direct Bluetooth HID emulation

### Non-Rooted Devices
- **Accessibility Service Mode**: Limited controller support via Android accessibility features
- **Some Games Supported**: Works with games that use standard Android gamepad input
- **No Root Required**: Enable accessibility service in device settings

**Automatic Detection**: App automatically detects device capabilities and switches between modes.

## Setup Instructions

### For Rooted Devices:
1. Install and grant root permissions when prompted
2. Pair your Nintendo Switch via Bluetooth
3. Launch Joy-Con Droid and select your controller type

### For Non-Rooted Devices:
1. Go to Settings → Accessibility → Downloaded Apps
2. Enable Joy-Con Droid accessibility service
3. Grant necessary permissions when prompted
4. Launch the app (limited functionality - mainly for Android games with gamepad support)

## Features

### Rooted Mode (HID Device)
✅ Full Joy-Con emulation (Left, Right, Pro Controller)
✅ Motion controls (Accelerometer & Gyroscope)
✅ Rumble feedback
✅ LED player indicators
✅ NFC/Amiibo support
✅ Full Nintendo Switch compatibility

### Non-Rooted Mode (Accessibility Service)
⚠️ Limited to Android app gamepad input
⚠️ No Nintendo Switch support
⚠️ Basic button mapping only
⚠️ No motion/rumble support
